Automated Follow Spot System Guide

3×3 Grid Room Calibration (Setup)

  1. Enter Calibration Mode: Click "Enter Calibration Mode" button
  2. Position UWB Tag: Place your UWB tag at one of the 9 grid positions on the floor
  3. Set Spotlight: Use the DMX sliders to point the spotlight at the tag's exact location
  4. Capture Point: Click the grid position button to record the position
  5. Repeat: Do this for at least 3 different positions (more = better accuracy)
  6. Calculate: Click "Calculate Room Calibration" when ready
  7. Test: Use "Test UWB Tracking" to verify the system works
  8. Follow Mode: Enable "Room Follow Mode" for automatic tracking

How It Works

  • UWB Tracking: Anchors at the front of the room track the performer's tag position
  • Grid Mapping: 3×3 grid creates a coordinate system for the stage area
  • Real-time Conversion: UWB coordinates are converted to pan/tilt angles
  • Automatic Following: Light automatically follows the performer within the calibrated area
  • Manual Override: Switch to manual mode for direct control when needed

Control Modes

  • Manual Control: Direct DMX control with sliders - UWB tracking disabled
  • UWB Tracking: Automatic performer following using calibration data
  • Calibration Mode: Setup mode for creating the 3×3 grid mapping

Tips for Best Results

  • Cover the Stage: Use positions that cover different areas of your performance space
  • Accurate Pointing: Make sure the spotlight is precisely aimed at the UWB tag during calibration
  • Stable Position: Keep the UWB tag still while capturing each calibration point
  • More Points: 5-9 points will give you much better accuracy than just 3
  • Test Movement: After calibration, test by moving the tag around to see if the light follows
