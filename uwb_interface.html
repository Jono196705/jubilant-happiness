Calibration Guide

Quick Start

For best results, use Point & Learn Calibration:

  1. Go to Calibration Page
  2. Place UWB tag at center of room
  3. Manually point spotlight at tag
  4. Click "Capture Point"
  5. Repeat for 3-5 different positions
  6. Click "Calculate Calibration"

Important Settings

  • Tag Height: Set correct height above ground
  • Light Orientation: Floor (upright) or Ceiling (hanging)
  • Anchor Distance: Measure actual distance between anchors

Troubleshooting

  • Poor Accuracy: Use more calibration points
  • Wrong Direction: Check pan/tilt scale values
  • Light Not Moving: Verify DMX connections
